# Azraq Announcement Studio

A self-contained announcement builder for Azraq. Edit every sentence, upload photos, watch the live animated preview, and download a 5 second looping GIF or a high resolution PNG. Everything runs in your browser: no install, no server, no data leaves your machine.

## Use it

- **Online:** open the GitHub Pages link for this repo (Settings > Pages) once enabled.
- **Offline:** download `Azraq_Announcement_Studio.html` and double-click it.

## How to use

1. Edit the kicker, headline, intro, people, and footer in the left panel. Wrap words in `*asterisks*` to color them Azraq Blue.
2. Upload photos. A transparent PNG gets the 3D pop-out effect; any other photo is framed inside the circle. Ready-made cutouts are in `photo-cutouts/`.
3. Pick GIF size and smoothness, then click **Download GIF (5s loop)** or **Download still PNG (2x)**.

Built with the Azraq brand system: navy #001528, Azraq Blue #117ACA, Space Grotesk and Inter.
